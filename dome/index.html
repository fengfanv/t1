t1 - 网页弹窗插件

这是一款面向前端初学者的网页弹窗，只需具备简单的JavaScript基础知识就能修改成属于自己的插件。

关注我
说明
1、引入

<script src="../js/t_alert.js"></script>

2、使用alert方法

t.alert(

"显示内容"

,function(){

alert("单击确定按钮执行的回调方法")

})

*回调方法选填

2、使用confirm方法

t.confirm(

"显示内容"

,function(){

alert("单击确定按钮执行的回调方法")

}

,function(){

alert("单击取消按钮执行的回调方法")

})

*回调方法选填

3、使用loadimg加载过渡窗口方法

t.loadimg()

*显示loadimg加载过渡窗口按钮已做打开3秒后关闭处理

4、关闭alert、confirm、loadimg、alertimg窗口方法

t.closeall()

*关闭t组件打开的所有弹层

5、显示图片弹层窗口方法

t.alertimg(

{

"imgurl":["../images/c.png"] /*图片地址*/

}

)
